TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Q: "git remote already exists" error
A: Run this first:
   git remote remove origin

Q: Backend deployment fails
A: Check Render dashboard → select service → Logs tab
   Look for error messages and fix

Q: Frontend can't connect to backend
A: 1. Check backend is running:
      curl https://jee-rank-backend.onrender.com/api/health
   2. Check frontend/app.js line 8 has production URL

Q: Port already in use locally
A: Run:
   lsof -i :8080
   Kill the process or use different port
